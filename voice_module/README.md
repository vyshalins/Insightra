# voice_module

Initial scaffold for transcription and emotion detection workflow.

## Structure

- `app/main.py` - FastAPI entry point (later)
- `app/transcribe.py` - Whisper logic (step 1)
- `app/emotion.py` - Emotion detection logic (step 2)
- `app/utils.py` - Optional helpers
- `samples/sample.wav` - Test audio placeholder
